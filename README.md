# Curso: Django + React Native - To-Do List App

Curso guiado por sesiones para aprender **Django** (Python) y **React Native** (Expo) construyendo una aplicacion To-Do List completa.

## Stack tecnologico

| Componente | Tecnologia | Version |
|---|---|---|
| Backend | Django | 6.0.x |
| API | Django REST Framework | - |
| Base de datos | PostgreSQL | - |
| Frontend mobile | React Native | 0.84.x |
| Plataforma mobile | Expo SDK | 55 |
| Autenticacion | JWT (simplejwt) | - |

## Estructura del proyecto

```
curso-django-react-native/
├── django/              # Backend - API REST con Django
├── react-native/        # Frontend mobile con Expo
└── programa.md          # Programa general del curso
```

## Setup - Django

```bash
cd django
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
